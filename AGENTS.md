# Codex Project Notes

## Project

This repository contains the static personal site for `denisyuce.com`.

## Structure

- `index.html` is the main site file and contains the HTML, CSS, and client-side JavaScript.
- `CNAME` configures the GitHub Pages custom domain.

## Development

- Keep the site static unless a larger migration is explicitly requested.
- Preserve the Russian-language content and existing single-file style when making small edits.
- Test visual changes by opening `index.html` in a browser or serving the folder locally.
